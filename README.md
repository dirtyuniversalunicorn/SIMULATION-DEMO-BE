# SIMULATION-DEMO-BE

Please, follow the steps below to run the application locally.

### Prerequisites
- Node.js
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dirtyuniversalunicorn/SIMULATION-DEMO-BE.git
2. Run
   ```bash
   npm install
3. Run
   ```bash
   npx prisma generate
4. Before running the application, you need to set up your environment variables.
   - Create a `.env` file in the root of the project.
   - Add the required environment variables.

> ⚠️ The environment variables were shared *via* **one time** link from the service https://onetimesecret.com/.
> In case something went wrong, feel free to reach out to me: jaroslavfilo95@gmail.com
   
5. Start the server. Server will start on port 3000.
   ```bash
   node index.js

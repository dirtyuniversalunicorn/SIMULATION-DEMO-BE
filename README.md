# SIMULATION-DEMO-BE

## Setup

Follow the steps below to run the application locally.

### Prerequisites
- Node.js (recommended: LTS version)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
2. Run
   ```bash
   npm install
3. Run
   ```bash
   npx prisma generate
4. Start the server. Server will start on port 3000
   ```bash
   node index.js

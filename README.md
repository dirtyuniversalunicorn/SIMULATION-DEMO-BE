- Node.js

   ```bash
   git clone https://github.com/dirtyuniversalunicorn/SIMULATION-DEMO-BE.git
2. Navigate to the directory:
3. ```bash
   cd SIMULATION-DEMO-BE
4. Run
   ```bash
   npm install

   npx prisma generate
